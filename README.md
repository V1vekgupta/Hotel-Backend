# Hotel-Backend
A secure, scalable hotel booking backend application developed from scratch using Java, Spring Boot, Spring Security, JWT, and MySQL.
This project provides RESTful APIs that allow users to browse hotel rooms, make bookings, and view reservation history, while admins can manage rooms and bookings securely.

**Key highlights**:

1. Secure authentication using JWT

2. Role-based authorization (Admin / User)

3. RESTful API design

4. MySQL-backed persistent storage

5. Clean layered architecture

| Category   | Technologies                |
| ---------- | --------------------------- |
| Language   | Java                        |
| Framework  | Spring Boot                 |
| Security   | Spring Security, JWT        |
| Database   | MySQL                       |
| ORM        | Spring Data JPA (Hibernate) |
| API Style  | RESTful                     |
| Build Tool | Maven                       |
